# RafuShop 

Mi eCommerce hecho en React para la entrega final de Talento Lab. Tiene catálogo de productos con búsqueda y paginación, carrito de compras, login/registro con Firebase y un panel de administración para gestionar los productos (crear, editar y borrar).

# Codigos de cupones de descuento 

- BIENVENIDA10 = 10% de descuento
- RAFU5000 = $5.000 de descuento fijo
- VENCIDO20 = vencido el 2024-01-01  (sirve para probar el mensaje de error de cupón vencido)
